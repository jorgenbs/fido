# Future Features

Ideas and enhancements captured for later consideration.

## Claude Code Slash Command Companion

Install a `~/.claude/commands/fido.md` slash command so developers can type
`/fido <issue-id>` inside an already-open Claude Code session to load the error
context without going through the terminal CLI.

This would let the fix workflow start directly from within an active Claude
session rather than requiring `fido fix` to launch a new one.
