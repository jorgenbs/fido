## Summary

First working version of Fido — an error triage sidecar that scans Datadog for error issues, generates structured reports, and enables AI-powered investigation and fix workflows.

## Highlights

- Datadog error issue scanning with configurable services and time window
- Structured report storage (`~/.fido/reports/<issue-id>/`)
- Error report generation with stack traces, logs, and Datadog links
- AI-powered investigation and fix commands via Claude
- GitLab CI status tracking for fix branches
- REST API and web dashboard
- Playwright-based frontend verification
