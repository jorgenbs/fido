## Summary

Full GitLab CI integration and dashboard quality-of-life improvements. CI status is now tracked per issue, and the dashboard surfaces investigation metadata for filtering and triage.

## Highlights

- GitLab CI status badges in dashboard and issue detail
- CI failure log fetching for `--iterate` fix attempts
- Re-fix button with CI failure context
- Investigation tags: Confidence, Complexity, Code Fixable
- Dashboard filters by service, confidence, complexity, fixability
- Checkbox selection with bulk ignore/unignore
- Verbose Datadog debug logging
- Deferred trace/stack fetching (scan no longer fetches heavy context)
