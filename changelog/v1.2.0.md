## Summary

Major improvements to the issue detail page, live operation status, and selection UX. The detail page became a full investigation workspace with markdown rendering, MR status polling, and inline actions.

## Highlights

- Styled markdown viewer with collapsible code blocks
- Live MR + CI status polling on issue detail
- `GET /api/issues/{id}/mr-status` endpoint
- Running operation indicator (pulsing badges for Investigating/Fixing)
- SSE auto-reconnect when viewing an issue with active operation
- Shift+click range selection for issue checkboxes
- Fix command display and re-fix guard in UI
- Investigation tag normalization (title-case)
- Curl verification requirement added to CLAUDE.md
